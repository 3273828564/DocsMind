## Run Frontend

### Install dependencies

```
yarn
```

```
yarn dev
```

## Run Backend

you need a python environment

### Create virtual environment

```
cd api-src
python -m venv .venv
```

### Active virtual environment

windows

```
.venv\Scripts\activate
```

mac

```
. .venv/bin/activate
```

### Install dependencies

```
pip install -r requirements.txt
```

### Run Services

```
flask run --reload
```
